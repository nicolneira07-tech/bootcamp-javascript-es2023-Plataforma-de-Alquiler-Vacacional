## 🎯 Descripción
Aplicación web desarrollada con Programación Orientada a Objetos (POO) que permite gestionar propiedades de alquiler vacacional.

Incluye:
- Catálogo de propiedades
- Sistema de usuarios
- Operaciones CRUD
- Búsqueda y filtrado
