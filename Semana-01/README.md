## 🎯 Descripción
Esta ficha representa una propiedad dentro de una plataforma de alquiler vacacional, similar a Airbnb.  
Permite visualizar información relevante como servicios, precio y disponibilidad, además de interactuar mediante acciones como reservar o agregar a favoritos.

También incluye una interfaz moderna con:
- Diseño responsivo
- Modo claro y oscuro 🌙☀️
- Notificaciones dinámicas (toast)
- Estilos visuales modernos con degradados
